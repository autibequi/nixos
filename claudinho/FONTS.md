## Regra para Banners DinГўmicos

Quando o conteГәdo de uma linha Г© dinГўmico (ex: weather, data), **NUNCA** preencher manualmente com espaГ§os.
Usar funГ§ГЈo de padding que:
1. Strippa escape codes ANSI para contar largura visual
2. Preenche com espaГ§os atГ© a largura fixa da box
3. SГі entГЈo adiciona o caractere de borda direita

```bash

pad_line() {
  local border="$1" content="$2" width="${3:-48}"
  local visible; visible=$(echo -e "$content" | sed 's/\x1b\[[0-9;]*m//g')
  local vlen; vlen=$(echo -n "$visible" | wc -m)
  local pad=$(( width - vlen ))
  [[ $pad -lt 0 ]] && pad=0
  printf '    %s  ' "$border"
  echo -ne "$content"
  printf "%${pad}s%s\n" "" "$border"
}
```
