Texto ao lado do avatar deve ter no máximo 2-3 palavras por linha. Conteúdo real vai abaixo, fora do code block.

**Why:** Linhas longas ao lado do avatar quebram o alinhamento e ficam feias no terminal (texto transborda para fora do bloco).

**How to apply:** Ao usar avatar + texto lateral: só frases curtíssimas inline. Tudo que for conteúdo vai em parágrafo separado, abaixo do code block.
