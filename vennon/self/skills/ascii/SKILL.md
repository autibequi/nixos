# ASCII — Arte no Terminal

Skill de output visual direto no terminal. Nao precisa de browser.

- `ascii.md` — regras de box-drawing, expressoes do avatar, caracteres permitidos
- `design-system.md` — palette Catppuccin, tokens, espacamento, formatacao

- Renderizar avatar (GLaDOS expressions)
- Box-drawing (blocos ERRO/SUCESSO/ACAO/INFO)
- Diagramas ASCII simples
- Formatacao visual em monospace

## Quando NAO usar (→ webview)

- Mermaid, flowcharts interativos → `webview`
