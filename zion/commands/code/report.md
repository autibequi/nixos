# Report — Relatório Consolidado da Branch

Gera relatório completo da branch atual: objetos por camada + stats de adições/remoções + resumo narrativo em português. Pode salvar no Obsidian.

## Entrada
- `$ARGUMENTS`: `[--branch auto|<nome>] [--format terminal|chrome|obsidian]`

## Instruções

Ler e executar o skill `code/report`:

```
Skill: code/report
```

Argumentos passados: `$ARGUMENTS`
