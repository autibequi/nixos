# Inspection — Inspeção de Qualidade Leve

Inspeção estática leve da branch atual: error handling, nil checks, contratos BFF/front, response shapes. Output com ✅ / ⚠️ / 🔴 por arquivo. Mais rápido que pr-inspector.

## Entrada
- `$ARGUMENTS`: `[--repo monolito|bo|front|all] [--foco handlers|services|all]`

## Instruções

Ler e executar o skill `code/inspection`:

```
Skill: code/inspection
```

Argumentos passados: `$ARGUMENTS`
