# Flows — Diagrama de Fluxo no Chrome

Gera diagrama Mermaid da arquitetura de fluxo da branch atual (read path + write path) e renderiza no Chrome com tema Catppuccin dark.

## Entrada
- `$ARGUMENTS`: `[--repo monolito|bo|front|all] [--modo leitura|escrita|full]`

## Instruções

Ler e executar o skill `code/analysis/flows`:

```
Skill: code/analysis/flows
```

Argumentos passados: `$ARGUMENTS`
