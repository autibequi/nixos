# Route — BO Container ou Front Student

Cria ou modifica route num dos frontends.

## Entrada
- `$ARGUMENTS`: `bo: <descrição>` ou `front: <descrição>`

## Roteamento

| Prefix | Agente | Framework |
|--------|--------|-----------|
| `bo:` | Coruja | Vue 2 |
| `front:` | Coruja | Nuxt 2 |

Se não houver prefix → perguntar: "bo ou front?"

## Instruções

```
Agent subagent_type=<Agente> prompt="Execute o skill route para: <descrição>"
```
